Ads Management plugin helps you to save your advertisement script and to use on post and page using shortcode.


== Description ==
Ads Management plugin helps you to save your advertisement script and get specific shortcode for every script categories by different size, different sponsor and type to use in your post. so you can change the script any time to update in your unlimited posts!
Though we have tried our best to make it useful for you and we are using this plugin on different websites, please let us know if you find any issue that is need to be fix for batter usability. 


== Installation ==
1. You can download and install the Ads Management plugin through the built-in Wordpress plugin installer. Alternately, download the zip file then extract and upload the '/ads-management/' folder to your '../wp-content/plugins/' folder.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View the instructions page in your the Wordpress backend to see example of shortcode to use on your post and pages.
4. You can also save to script of your sponsor to show atthe top and at the bottom of Posts.
